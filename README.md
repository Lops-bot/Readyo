# Readyo 🗣️

An open source AAC (Augmentative and Alternative Communication) iOS app designed to help children with learning disabilities and speech delays communicate more effectively.

## The Problem
Millions of children worldwide experience speech delays or communication challenges. Their parents are warriors — showing up every day, navigating therapy, school systems, and uncertainty — often feeling very alone.

**Readyo exists to give these children a voice.**

## How It Works
1. 🎤 **Parent speaks** a question using their voice
2. 🖼️ **App converts** speech to picture symbols instantly
3. 👆 **Child taps** a picture to respond
4. 🔊 **App speaks** the full response out loud

## Features
- 🎤 Voice recognition for parents
- 🖼️ 200+ ARASAAC picture symbols
- 👍 Yes/No/Maybe responses for closed questions ("Do you want ice cream?")
- 🔤 Category picture grid for open questions ("What do you want to eat?")
- 🔊 Natural text to speech responses
- 📱 Works on iPhone and iPad
- ✈️ Fully offline — no internet required
- 🆓 Free and nonprofit

## Getting Started

### Requirements
- Mac with Xcode 15+
- iOS 17.0+
- iPhone or iPad for testing

### Installation
1. Clone the repository
```
git clone https://github.com/Lops-bot/Readyo.git
```
2. Open `Readyo.xcodeproj` in Xcode
3. Select your target device
4. Hit ▶️ Play!
