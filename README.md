# Readyo 🗣️

An AAC (Augmentative and Alternative Communication) iOS app designed to help children with learning disabilities communicate more effectively.

## What it does
- Parent speaks a question using voice
- App converts speech to keywords
- Keywords display as clear picture symbols
- Child taps a picture to respond
- App speaks the full response out loud

## Who it's for
- Children with learning disabilities
- Children with Autism
- Non-verbal or limited verbal children
- Parents and caregivers

## Features
- 🎤 Voice recognition for parents
- 🖼️ Picture-based communication (202 words)
- 👍 Yes/No/Maybe responses for closed questions
- 🔤 Category picture grid for open questions
- 🔊 Text to speech responses
- 📱 Works on iPhone and iPad
- ✈️ Fully offline

## Tech Stack
- Swift / SwiftUI
- Apple Speech Recognition
- AVFoundation
- ARASAAC pictograms (coming soon)

## Getting Started
1. Clone the repository
2. Open Readyo.xcodeproj in Xcode
3. Select your target device
4. Hit Play!
